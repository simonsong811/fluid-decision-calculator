# Fluid Decision Calculator

Bilingual research-use septic shock fluid dose-risk calculator.

## GitHub Pages

This repository is configured for GitHub Pages via GitHub Actions. After the first workflow run succeeds, the calculator will be available at:

https://simonsong811.github.io/fluid-decision-calculator/

## Notes

- Static single-page HTML calculator.
- Supports Chinese and English UI switching.
- Exploratory retrospective MIMIC-IV model for research visualization only; not for bedside orders.
